Install
------------------------------------------------------------------------------

``aws_ecs_devops`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install aws_ecs_devops

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade aws_ecs_devops
